内容传送编码是一种以任意8位字节列组合的描述形式，这种形式不容易被人直接识别。
经过base64编码后的数据会比原来数据长约1/3，经过base64编码后的字符串的字符数是以4为单位的整数倍。
实现base64的算法推荐使用apache的common codec类库。

2、应用场景
电子邮件传输、网络数据传输、密钥存储、数字证书存储。
